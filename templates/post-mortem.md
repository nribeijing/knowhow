## 🔍 根本原因分析

### 5 Why 分析法

```
1. Why: 为什么 API 返回 500 错误？
   → 因为数据库连接超时

2. Why: 为什么数据库连接超时？
   → 因为连接池耗尽

3. Why: 为什么连接池耗尽？
   → 因为有慢查询占用连接

4. Why: 为什么有慢查询？
   → 因为缺少索引

5. Why: 为什么缺少索引？
   → 因为代码审查时未发现，测试环境数据量小未暴露
```

### 根本原因

```
[总结根本原因]

例：
1. 直接原因：数据库缺少索引导致慢查询
2. 间接原因：测试环境数据量小，未暴露性能问题
3. 根本原因：代码审查清单缺少性能检查项
```
